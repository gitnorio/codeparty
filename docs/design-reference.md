# Design Reference

This file keeps the visual references used for the authenticated product shell and marketing pages.

## Source direction

- Primary inspiration: `https://blabla.ai/`
- Core visual cues:
  - off-white background
  - saturated violet primary accent
  - lavender secondary surfaces
  - very large editorial headings
  - oversized rounded corners
  - simple cards with lots of breathing room
  - gradient showcase panels

## Authenticated app shell brief

After login, the connected layout should use:

- Left sidebar:
  - Dashboard
  - Matchmaking
  - My Team
  - My Project
  - Public Profile
  - Settings
- Topbar:
  - notifications
  - user profile
  - logout
- Main content:
  - matchmaking status
  - current team
  - active project
  - GitHub repo
  - progress
  - contributions
